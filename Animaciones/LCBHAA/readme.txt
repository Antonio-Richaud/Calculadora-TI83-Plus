Instalación:
-------------

Para los usuarios de TI-83, copia LCBHAA.83P a tu calculadora usando tu software de enlace favorito.
Para los usuarios de TI-83+, copia LCBHAA.8XP a tu calculadora usando tu software de enlace favorito.

Requiere ION o un shell compatible para ejecutarse. Una vez que el archivo esté copiado en tu calculadora, simplemente ejecútalo
desde un shell compatible con ION.
